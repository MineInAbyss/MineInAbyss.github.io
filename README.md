# Mine in Abyss Wiki

Welcome to the Mine in Abyss Wiki. This is our main documentation site built on [socky-docs](https://github.com/0ffz/shocky-docs).
It links to other projects and their docs. You can find it at https://docs.mineinabyss.com, or contribute to it on [GitHub](https://github.com/MineInAbyss/MineInAbyss.github.io).

## Building locally

- `gradle docsServe`
- `gradle docsGenerate`
